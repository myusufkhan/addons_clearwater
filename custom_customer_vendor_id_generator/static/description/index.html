Custom Customer Supplier ID Generator

Generates Unique ID for Customers and Suppliers

After installation Customer ID field will be added to Customer form. On Creating a Customer it will automatically generate a unique code for Customer with the prefix of the first letter of Customer name.
Same with Supplier, Supplier ID field will be added to Supplier form. On Creating a Supplier it will automatically generate a unique code for Supplier with the prefix of the first letter of Supplier name.
For Customer Creation, A new sequence will be created for generating Customer ID.

For Supplier Creation, A new sequence will be created for generating Supplier ID.
